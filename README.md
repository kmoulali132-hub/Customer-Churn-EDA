# 📊 Customer Churn Analysis - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **Telco Customer Churn** dataset to identify customer behavior, understand churn patterns, and discover the key factors influencing customer attrition. The analysis provides meaningful insights through data visualization and statistical exploration, helping businesses improve customer retention strategies.

---

## 🎯 Objectives

- Calculate the overall customer churn rate.
- Explore customer demographics.
- Analyze customer tenure.
- Examine the relationship between churn and contract types.
- Analyze the impact of payment methods on churn.
- Visualize important trends using charts and graphs.
- Identify key features influencing customer churn.

## 📊 Key Findings

- Overall customer churn rate is **26.54%**.
- Customers using **Fiber Optic Internet** have a higher churn rate.
- Customers paying through **Electronic Check** are more likely to churn.
- Higher **Monthly Charges** are associated with increased churn.
- Customers with longer **tenure** are less likely to churn.
- Customers with **Two-Year Contracts** show the lowest churn rate.
- Paperless billing has a positive association with customer churn.
